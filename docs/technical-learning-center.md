## 1. 核心觀念：股票不是賭數字

股票的本質不是買一個會跳動的價格，而是買進公司的一部分所有權。

買股票代表你把錢從「消費者」轉成「生產者」，讓企業、工程師、管理層、品牌、技術、資產幫你工作。

股票主要賺兩種錢：

- 股息：像母雞下蛋，適合穩健、長期、現金流型投資。
- 資本增值：公司變強、獲利變好、股價上漲，適合成長型投資。

核心公式：

```text
總報酬 = 基本面成長 + 市場估值情緒 + 股息
```

短期股價容易受情緒影響，長期還是要回到公司價值。

基本面看「值不值得買」，技術面看「現在適不適合買」。

## 2. 看盤介面核心數字

打開股票 APP，不要只看漲跌，要先看懂這些數字。

| 名稱 | 意義 | 觀察重點 |
| --- | --- | --- |
| 現價 | 現在成交價格 | 是否接近支撐、壓力或均線 |
| 漲跌幅 | 今天相對昨天收盤漲跌多少 | 短線強弱與市場情緒 |
| 今開 | 今天開盤價 | 開高、開低與盤中承接 |
| 昨收 | 昨天收盤價 | 今日漲跌基準 |
| 最高 / 最低 | 當日價格活動範圍 | 判斷壓力、支撐與波動 |
| 成交量 | 有多少股被交易 | 資金參與程度 |
| 成交額 | 成交量乘以成交價 | 資金規模與熱度 |
| 量比 | 今日成交熱度相對過去平均 | 熱門與否 |
| 換手率 | 股票被交易的活躍程度 | 籌碼換手是否劇烈 |
| 市值 | 公司整體市場價值 | 大型股、中型股、小型股定位 |

最重要的判讀：

```text
價格代表方向，成交量代表力量。
```

價格可以騙人，但成交量比較難騙人。

## 3. K 棒是市場語言

一根 K 棒由四個價格組成：

- 開盤價
- 最高價
- 最低價
- 收盤價

K 棒不是只看紅色或綠色，而是看實體、影線、位置、趨勢。

### 實體

實體越長，代表攻擊力越強。

- 大陽線：多方強。
- 大陰線：空方強。

### 上影線

代表衝高後被賣下來，上方有壓力。

常見意義：

- 高檔長上影：追價力道不足，可能遇到賣壓。
- 突破後長上影：小心假突破。

### 下影線

代表殺低後被買上來，下方有支撐。

常見意義：

- 支撐附近長下影：有承接。
- 低檔長下影：可能有買盤進場。

### 特殊 K 棒

| 型態 | 意義 | 注意 |
| --- | --- | --- |
| 十字星 | 多空猶豫，可能變盤 | 要搭配位置與成交量 |
| 蜻蜓線 | 下影線長，可能有支撐 | 低檔較有參考性 |
| 墓碑線 | 上影線長，可能有賣壓 | 高檔較危險 |
| 吊人線 | 高檔長下影，可能警訊 | 隔日確認很重要 |
| 錘子線 | 低檔長下影，可能止跌 | 需放量或隔日轉強 |

K 棒口訣：

```text
不要只看顏色，要看實體、影線、位置、趨勢。
```

## 4. 均線 MA：趨勢方向的濾鏡

均線是過去一段時間價格的平均值，用來把雜訊變平滑。

常見均線：

| 均線 | 常見用途 |
| --- | --- |
| MA5 | 短線情緒線，強勢股常沿著 MA5 走 |
| MA10 | 短線趨勢延續 |
| MA20 / MA30 | 月線，中期生命線與支撐觀察線 |
| MA50 / MA60 | 中期趨勢 |
| MA200 | 長期趨勢 |

### 五日線戰法

強勢股通常有這些特徵：

- 股價站在五日線上。
- 拉回不破五日線。
- 跌破後快速站回。
- 五日線持續向上彎。
- 成交量溫和放大。

注意：看到碰五日線不能直接買，要等止跌訊號，例如下影線、紅 K 反包、量縮。

五日線口訣：

```text
沿著五日線走的股票，通常還沒走完。
跌破五日線不可怕，站不回去才危險。
```

### 月線支撐

月線通常用 MA20 或 MA30 觀察。

月線支撐有效時，常見特徵：

- 股價跌到月線附近。
- 成交量縮小。
- 出現下影線。
- 出現止跌紅 K。
- 反彈後重新站穩。

月線支撐在上升趨勢中比較可靠。如果本身是下跌趨勢，月線支撐不一定有效。

月線口訣：

```text
支撐不是保證不跌，而是要看跌到支撐後市場怎麼反應。
```

## 5. 支撐與壓力：市場人性的價格記憶

### 支撐

支撐就是股價跌到某個位置，容易止跌反彈。

常見支撐來源：

- 前波低點。
- 主力成本區。
- 大量成交區。
- 均線附近。
- 箱型底部。

支撐附近常見訊號：

- 下影線。
- 成交量縮小。
- 止跌紅 K。
- 爆量反彈。

### 壓力

壓力就是股價漲到某個位置，容易漲不動。

常見壓力來源：

- 前高。
- 套牢區。
- 大量成交區。
- 主力出貨區。
- 箱型頂部。

壓力附近常見訊號：

- 長上影線。
- 爆量不漲。
- 開高走低。
- 無法突破前高。

最重要觀念：

```text
支撐跌破後，可能變成新壓力。
壓力突破後，可能變成新支撐。
```

## 6. 量價關係：技術分析的核心

量價關係最重要一句話：

```text
量在價先，有量才有價。
```

### 價漲量增：健康上漲

股價上漲，成交量也放大，代表資金進場、市場追價意願強。

這是最健康的上漲型態。

### 價漲量縮：小心假強勢

股價漲，但成交量越來越少，代表追價的人變少，上漲動能可能減弱。

容易出現假突破。

### 價跌量增：危險訊號

股價跌，成交量放大，代表賣壓增加、恐慌增加、主力可能出貨。

高檔出現尤其危險。

### 價跌量縮：跌勢可能趨緩

股價跌，但成交量縮小，代表賣壓開始變小，有機會止跌整理。

### 低檔爆量：可能是機會

股票跌很久後突然爆大量、長紅 K，可能代表主力開始吃貨。

但還要看是否站穩關鍵位置。

### 高檔爆量：可能是出貨

股票大漲後高檔爆歷史大量，伴隨長上影線，很多時候代表主力賣給最後追高的人。

量價口訣：

```text
低檔看量找機會，高檔爆量要小心。
成交量是主力留下最誠實的腳印。
```

## 7. 型態學：找買點與賣點

型態不能只看像不像，要看有沒有突破或跌破關鍵線。

### 偏多型態

| 型態 | 意義 | 確認方式 |
| --- | --- | --- |
| 頭肩底 / 倒頭肩 | 底部反轉型態 | 突破頸線後偏多 |
| W 底 / 雙底 | 兩次測底不破 | 突破頸線後偏多 |
| 上升旗形 | 急漲後整理 | 突破旗面後續漲 |
| 上升三角形 | 高點持平、低點墊高 | 放量突破壓力後偏多 |
| 三重底 | 多次測底不破 | 突破頸線後偏多 |
| 咖啡杯型態 | 長時間圓弧整理後突破 | 突破杯緣與把手整理區 |

### 偏空型態

| 型態 | 意義 | 確認方式 |
| --- | --- | --- |
| M 頭 / 雙頂 | 兩次攻高失敗 | 跌破頸線後偏空 |
| 菱形頂 | 高檔整理後轉弱 | 跌破整理區後偏空 |
| 下跌旗形 | 急跌後反彈整理 | 跌破旗面後續跌 |
| 三重頂 | 三次攻高失敗 | 跌破頸線後偏空 |
| 倒 V 型反轉 | 急漲後快速轉跌 | 情緒退潮，需避開追高 |

型態口訣：

```text
型態不能只看像不像，要看有沒有突破或跌破關鍵線。
```

### 7.1 十種圖表形態

這一組圖表形態可以併入型態學一起看。重點不是背圖形，而是理解它們背後的支撐、壓力、多空力量與市場情緒。

| 型態 | 常見方向 | 實戰重點 |
| --- | --- | --- |
| 頭肩形態 | 頭肩頂偏空，頭肩底偏多 | 由左肩、頭部、右肩組成，核心是頸線是否被跌破或突破 |
| 雙頂 | 偏空反轉 | 兩次攻高失敗，跌破支撐或頸線後容易轉弱，也就是 M 頭 |
| 雙底 | 偏多反轉 | 兩次測底不破，突破上方壓力後代表空方力道可能衰退，也就是 W 底 |
| 圓底 | 偏多反轉或延續 | 股價從下跌、止跌、整理到慢慢轉強，最好等突破壓力後確認 |
| 杯柄形態 | 偏多延續 | 杯身像圓底，杯柄是短暫回調，突破杯柄壓力後才算確認 |
| 楔形形態 | 上升楔形偏空，下降楔形偏多 | 價格被兩條傾斜趨勢線夾住且空間變窄，不能只看方向表面 |
| 三角旗形 / 旗形 | 延續或轉向 | 明顯漲跌後的小幅整理，要等突破方向確認 |
| 上升三角形 | 偏多延續 | 上方高點接近水平，下方低點逐漸墊高，突破水平壓力後偏多 |
| 下降三角形 | 偏空延續 | 下方低點接近水平，上方高點越來越低，跌破支撐後偏空 |
| 對稱三角形 | 雙邊，常偏延續 | 高點越來越低、低點越來越高，若前面有明確趨勢通常偏延續，否則等市場選邊 |

### 7.2 實戰分類

反轉型態：原本方向可能要變了。

- 頭肩形態。
- 雙頂。
- 雙底。
- 圓底。
- 楔形。

反轉型態實戰重點：

```text
不要猜最高最低，要等頸線、支撐、壓力被確認突破或跌破。
```

延續型態：整理後可能繼續原本方向。

- 杯柄。
- 旗形。
- 上升三角形。
- 下降三角形。
- 對稱三角形。

延續型態實戰重點：

```text
先看前面有沒有明確趨勢，再看整理期間有沒有量縮，突破時有沒有放量。
```

雙邊型態：還不知道方向，要等市場選邊。

- 對稱三角形。
- 三角旗形。

雙邊型態實戰重點：

```text
不要預設方向，等突破或跌破後再判斷。
```

### 7.3 型態判斷問題清單

看到任何型態時，先問自己：

- 這是反轉還是延續？
- 上方壓力在哪？
- 下方支撐在哪？
- 突破有沒有量？
- 收盤有沒有站穩？
- 突破後有沒有續強？
- 如果失敗，我的停損在哪？

型態整合口訣：

```text
型態只是地圖，不是答案。
型態不是買點，突破確認才是買點；形狀只是提示，量價確認才是答案。
```

## 8. 延續型態：趨勢中的休息

延續型態不是反轉，而是趨勢中途休息。

### 上升旗形

旗桿向上，旗面向下整理，突破旗面後偏多。

口訣：

```text
旗桿向上，旗面整理後突破。
```

### 下跌旗形

旗桿向下，旗面向上整理，跌破旗面後偏空。

口訣：

```text
旗桿向下，旗面整理後再跌。
```

### 上升三角形

高點持平，低點上升，代表買方越來越強。

口訣：

```text
高點持平，低點上升，整理後突破。
```

### 下跌三角形

低點持平，高點下降，代表賣壓越來越重。

口訣：

```text
低點持平，高點下降，整理後跌破。
```

### 對稱三角形

高點下降、低點上升，多空壓縮，等方向突破。

延續型態口訣：

```text
整理不是結束，突破方向才是答案。
```

## 9. 突破前的盤整型態

真正飆股常常是：

```text
先盤整 -> 再突破 -> 最後大漲
```

盤整是主力布局的過程。

主力盤整的目的：

- 慢慢吸籌。
- 消化賣壓。
- 洗掉沒耐心的人。
- 降低成本。
- 為突破做準備。

常見盤整型態：

| 型態 | 意義 | 觀察重點 |
| --- | --- | --- |
| 箱型整理 | 上有壓力，下有支撐 | 突破箱頂且放量，可能進入主升段 |
| 三角收斂 | 高點越來越低，低點越來越高 | 空間壓縮，突破後容易爆發 |
| 平台整理 | 大漲後橫盤，量縮 | 五日線慢慢跟上後再放量，可能再攻一波 |

盤整時間與突破力道：

| 盤整時間 | 可能行情 |
| --- | --- |
| 1 到 2 週 | 小整理，小行情 |
| 2 到 4 週 | 中整理，中波段 |
| 1 到 3 個月以上 | 大整理，可能大波段或主升段 |

盤整口訣：

```text
盤整是主力的沉默，突破是市場的爆發。
```

## 10. 真突破 vs 假突破

判斷突破要看四個字：

```text
量、價、勢、守
```

- 量：有沒有成交量。
- 價：收盤有沒有站穩。
- 勢：突破後有沒有續強。
- 守：拉回有沒有守住。

### 真突破特徵

- 成交量明顯放大。
- 收盤站穩壓力區。
- 突破後不容易跌回原區間。
- 股價快速轉強續漲。
- 拉回守住關鍵支撐。

### 假突破特徵

- 突破沒量。
- 只有盤中突破，收盤沒站穩。
- 長上影線，賣壓重。
- 突破後隔天沒續強。
- 突破後馬上跌回區間。
- 高檔爆量突破，小心主力出貨。

突破口訣：

```text
突破不是看衝過去，而是看站得穩不穩。
```

## 11. 主力洗盤

洗盤是主力故意製造恐慌，把沒耐心的人洗出去。

目的：

- 降低未來拉抬時的賣壓。
- 震掉浮額。
- 收集更便宜籌碼。
- 讓籌碼更乾淨。

常見洗盤方式：

| 洗盤方式 | 說明 |
| --- | --- |
| 假跌破洗盤 | 故意跌破支撐、均線、前低，讓散戶以為崩盤，但很快拉回 |
| 橫盤磨人洗盤 | 股價不大漲不大跌，天天震盪，讓散戶沒耐心 |
| 急殺長黑洗盤 | 用急跌嚇人，如果後面沒續破底，可能是假跌真洗盤 |
| 拉高後快速回落 | 吸引追價，再快速壓回，套住追高者 |
| 利空消息洗盤 | 利用新聞或市場恐慌，讓散戶賣出 |

真洗盤常見特徵：

- 跌破後快速拉回。
- 成交量沒有失控。
- 守住中期均線或主力成本區。
- 籌碼沒有明顯鬆動。
- 後續重新轉強。

洗盤口訣：

```text
洗盤不是行情結束，而是主力在清理道路。
```

## 12. 技術指標中心

技術指標是輔助工具，不是答案。真正的決策要看位置、趨勢、量價、型態與風險報酬。

### 12.1 布林通道 Bollinger Bands

布林通道由三條線組成：

- 中線：20 日 SMA。
- 上軌：SMA20 + 2 個標準差。
- 下軌：SMA20 - 2 個標準差。

標準差代表波動率。

布林通道看什麼：

- 通道變寬：波動變大，行情劇烈。
- 通道變窄：波動變小，正在壓縮。
- Squeeze 窄通道：常暗示未來可能突破。

兩種主要用法：

1. 反轉觸及邊界：碰上軌可能偏熱，碰下軌可能偏冷，但不能單獨當買賣訊號。
2. 延續騎乘趨勢：強勢股可能沿著中線與上軌往上跑，碰上軌不一定要賣。

布林口訣：

```text
布林通道不是碰上軌就賣、碰下軌就買，要先判斷是盤整還是趨勢。
```

### 12.2 MACD

MACD 不是神，也不是買賣點本身。

MACD 用來看：

- 趨勢方向。
- 多空力道。
- 上漲是否加速。
- 下跌是否減弱。
- 主力節奏變強還是變弱。

MACD 主要有三個部分：

| 名稱 | 意義 |
| --- | --- |
| DIF 快線 | 反應短線情緒 |
| MACD / DEA 慢線 | 反映中期方向 |
| 柱狀體 OSC | 反映多空力道變化 |

黃金交叉：DIF 往上突破慢線，代表多方力量可能轉強。

有效黃金交叉通常具備：

- 低檔出現。
- 股價同步轉強。
- 成交量增加。
- MACD 翻紅。
- 均線開始上彎。

死亡交叉：DIF 往下跌破慢線，代表短線轉弱。

危險死亡交叉常見於：

- 高檔。
- 爆量長黑。
- 跌破重要均線。
- MACD 翻綠。
- 反彈越來越弱。

柱狀體重點：

- 紅柱變長：多頭加速。
- 紅柱縮短：上漲動能減弱。
- 綠柱變長：空方加速。
- 綠柱縮短：空方力道減弱，可能接近底部。

MACD 背離：

- 頂背離：股價創新高，但 MACD 沒創新高，代表漲勢變弱。
- 底背離：股價創新低，但 MACD 沒再創低，代表空方力道衰退。

MACD 口訣：

```text
MACD 不是買點，位置才是買點。
真正高手看 MACD，不是只看金叉死叉，而是看主力力道變強還是變弱。
```

### 12.3 RSI

RSI 用來觀察相對強弱與過熱過冷。

常見設定：

- RSI 14：常用短中期強弱。
- RSI 9 / RSI 55：可用來觀察短線強弱與中期相對趨勢。

常見判讀：

| RSI 區間 | 解讀 |
| --- | --- |
| 70 以上 | 過熱，但強勢股可能持續鈍化 |
| 50 以上 | 多方較有優勢 |
| 50 以下 | 空方較有優勢 |
| 30 以下 | 過冷，但弱勢股可能繼續弱 |

RSI 注意事項：

- 不能看到 70 就賣，強勢股常常 RSI 高檔鈍化。
- 不能看到 30 就買，弱勢股可能持續破底。
- RSI 背離要搭配型態、支撐、成交量確認。

### 12.4 KD

KD 用來看短線動能與轉折。

常見判讀：

- K 上穿 D：短線轉強。
- K 下穿 D：短線轉弱。
- KD 高檔鈍化：強勢趨勢延續，但追高風險增加。
- KD 低檔鈍化：弱勢趨勢延續，不能太早抄底。

KD 應搭配：

- 均線方向。
- 股價位置。
- 成交量。
- 是否突破或跌破關鍵位置。

### 12.5 ATR

ATR 是平均真實波幅，用來衡量波動，不是方向指標。

常見用途：

- 設定停損距離。
- 設定移動停損。
- 衡量波動風險。
- 避免停損太近被正常震盪洗掉。

範例：

```text
移動停損 = 最高價 - ATR x 倍數
移動停利 = 碰到初始停利後，跟隨股價或高點往上調整
```

ATR 核心：

- ATR 越大，代表波動越大，停損距離要更寬。
- ATR 越小，代表波動越小，停損距離可較窄。
- ATR 不是買賣訊號，主要是風控工具。

### 12.6 HMA

HMA 是 Hull Moving Average，目標是比一般均線更靈敏地反映趨勢。

常見用途：

- 觀察短線趨勢轉折。
- 判斷價格是否重新站上趨勢線。
- 搭配標準差或 ATR 設定停損停利。

HMA 策略常見概念：

- 收盤價上穿 HMA 且收盤價大於 HMA，視為關鍵 K。
- 下一根 K 作為進場觀察。
- 需搭配成交量過濾，避免無量股票。

### 12.7 Standard Deviation 標準差

標準差衡量價格波動程度。

常見用途：

- 當作停損距離的波動基準。
- 當作移動停利的調整幅度。
- 搭配 HMA 找趨勢與風控位置。

範例：

```text
停損 = 進場價 - 標準差 x N
移動停利 = 進場價 + 最新標準差 x N
```

## 13. 基本面與技術指標

### 估值類：看貴不貴

| 指標 | 意義 |
| --- | --- |
| PE 本益比 | 用目前股價買進後，靠利潤回本需要幾年 |
| PB 股價淨值比 | 股價相對公司淨資產 |
| PS 市銷率 | 市值相對營收 |
| PEG 成長型本益比 | PE 除以預期成長率 |

估值不能單看數字，要跟同產業、歷史區間比較。

### 盈利類：看會不會賺錢

| 指標 | 意義 |
| --- | --- |
| EPS | 每股盈餘 |
| ROE | 股東權益報酬率 |
| 毛利率 | 產品或服務的利潤空間 |
| 淨利率 | 最後真正留下的利潤比例 |

重點是公司不只要營收高，還要能把錢留下來。

### 安全類：看穩不穩

| 指標 | 意義 |
| --- | --- |
| 資產負債率 | 財務槓桿與負債壓力 |
| 流動比率 | 短期償債能力 |
| 自由現金流 FCF | 公司可自由運用的現金 |
| 股息率 | 現金回饋能力 |

重點是公司不能只是會賺，還要能活得久。

### 趨勢類：看現在能不能買

| 指標 | 用途 |
| --- | --- |
| MA 均線 | 看趨勢方向 |
| MACD | 看多空力道 |
| RSI | 看強弱與過熱過冷 |
| Volume 成交量 | 確認真假突破與資金參與 |

核心口訣：

```text
基本面看值不值得買，技術面看現在適不適合買。
```

## 15. 實戰 SOP

### 第一步：先看大方向

現在是上升趨勢、下跌趨勢，還是盤整？

觀察：

- 均線方向。
- 股價是否站上均線。
- 高低點是否墊高。
- 成交量是否配合。

### 第二步：找位置

現在股價在哪？

- 靠近支撐？
- 靠近壓力？
- 剛突破壓力？
- 跌破支撐？
- 在箱型內？
- 在高檔爆量？
- 在低檔止跌？

提醒：

```text
位置比指標重要。
```

### 第三步：看量價

有沒有量？

- 上漲有沒有放量？
- 拉回有沒有量縮？
- 突破有沒有量增？
- 高檔有沒有爆量長上影？
- 下跌有沒有放量？

量價是判斷真假的核心。

### 第四步：看型態

有沒有：

- W 底。
- M 頭。
- 頭肩底。
- 上升旗形。
- 下跌旗形。
- 箱型整理。
- 三角收斂。
- 平台整理。

型態一定要等確認，不要看到像就衝。

### 第五步：用指標輔助

- 用 MA 看趨勢。
- 用 MACD 看多空力道。
- 用 RSI 看過熱過冷。
- 用 KD 看短線動能。
- 用布林通道看波動率。
- 用成交量確認真假突破。
- 用 ATR 控制停損距離。

### 第六步：設定停損

任何交易前都要先知道：

```text
如果看錯，在哪裡出場？
```

常見停損位置：

- 跌破支撐。
- 跌破頸線。
- 跌破五日線後站不回。
- 跌破月線且收盤失守。
- 突破失敗跌回區間。

### 第七步：控制資金

用 2% 原則：

```text
單筆最大風險不要超過總資金 2%。
```

不是先決定買多少，而是先決定停損在哪，再倒推能買多少。

部位計算：

```text
可承受虧損金額 = 總資金 x 單筆風險比例
每股風險 = 買進價 - 停損價
可買股數 = 可承受虧損金額 / 每股風險
```

## 16. 交易前檢查表

進場前至少確認：

- 大盤環境是否偏多或偏空。
- 股價是否在合理位置，不是盲目追高。
- 成交量是否支持突破或轉強。
- 均線是否轉強或至少不明顯空頭排列。
- 型態是否完成確認。
- 停損位置是否明確。
- 風險報酬比是否合理。
- 單筆風險是否控制在可承受範圍。
- 是否有重大新聞、法說、財報、除權息或公告風險。

進場後追蹤：

- 是否守住原本設定的支撐。
- 是否放量續強。
- 是否出現高檔爆量長上影。
- 是否跌破關鍵均線。
- 是否需要上調移動停損。
- 是否達到停利規劃。

## 17. 給 AI 系統使用的技術判讀規則

這一節可作為系統未來產生「個股分析、觀察報告、AI 策略說明」的共同規則。

### 偏多條件

可視為偏多的條件：

- 股價站上 MA20。
- MA5 > MA10 > MA20。
- MA20 向上。
- 成交量大於近期平均。
- 突破壓力且收盤站穩。
- MACD DIF 在 MACD 線上方，OSC 轉強。
- RSI 站上 50 且未明顯背離。
- KD 黃金交叉且位置不過度追高。

### 轉弱條件

可視為轉弱的條件：

- 股價跌破 MA20 且站不回。
- MA5 下彎並跌破 MA10。
- 放量長黑。
- 高檔爆量長上影。
- MACD 死亡交叉且 OSC 翻綠。
- RSI 跌破 50。
- 跌破支撐或頸線。

### 觀望條件

應避免積極進場的情況：

- 成交量太低，流動性不足。
- 價格在壓力附近但沒有放量突破。
- 指標互相矛盾。
- 大盤環境明顯偏空。
- 個股連續急漲後乖離過大。
- 重大事件公布前風險不明。

### 風險語句模板

個股分析可以用這類語句保持一致：

```text
支撐不是保證不跌，跌到支撐後是否出現承接才是重點。
突破不是看盤中衝過去，而是看收盤能否站穩。
成交量不足時，訊號可信度需打折。
若跌破防守價，原本偏多假設失效，應重新評估。
```

## 18. 最後總口訣

1. 股價是表面，成交量是真相。
2. 支撐附近找機會，壓力附近懂風險。
3. 突破不是看衝過去，而是看站不站得穩。
4. 洗盤不是行情結束，而是主力清理道路。
5. MACD 不是買點，位置才是買點。
6. 均線不是水晶球，只是趨勢濾鏡。
7. 高檔爆量要小心，低檔爆量看機會。
8. 交易核心是截斷虧損，讓利潤奔跑。
9. 策略 + 紀律 + 耐心，才是長期穩定獲利的核心。
